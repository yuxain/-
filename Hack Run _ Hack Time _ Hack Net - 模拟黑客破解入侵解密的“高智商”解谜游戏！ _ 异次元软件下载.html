话说在 iPad 和手机上，我对解谜/解密游戏都情有独钟，毕竟这是最适合触摸屏设备游玩的一种类型，而且这些游戏往往考验智商和想象力，通关后会有很大的成就感。

我们刚介绍过 The Room 密室 和 Blek 手指画线，而今天推荐的则是“高智商”玩家的专属游戏——黑客入侵解谜游戏系列！你将化身为强大的黑客，接受各种任务，譬如入侵别人的邮箱账号窃取重要信息、破解密码等。游戏以类似DOS界面的命令行方式进行，看上去很酷很高技术，实际上玩家只需输入一些基础简单的指令即可一步步解开谜团……

富有挑战性和趣味的黑客解密游戏：

目前这个黑客解谜游戏系列已经推出了 Hack Run、Hack Run Zero、Hack Time、Hack NET 共4款作品，游玩顺序应该也是这样了。游戏的难度其实并不是很高，开始的前面几关基本上是相当于教学关，逐渐展开游戏的剧情。

虽然游戏界面看起来很像编程写代码的界面或者说是命令行终端，很酷很黑客Feel (题外话：黑客装逼神器)，但游戏的命令操作并不繁琐，实际上并不需要太多真正的电脑技术即可进行。

[ Hack RUN iPhone版截图 ]

这个黑客系列的游戏很赞，可以让无数向往技术的朋友体验一把当黑客的感觉（当然是模拟的），相信很多小伙伴也曾都幻想过自己有一天也能成为神一般存在的黑客的吧！

[ Hack Time HD iPad版游戏截图 ]

可惜的是，游戏全程都是英文的，所以英语不够 OK 的同学还是有点难度。如果你英文基础还好，这个系列游戏相信将会是你玩过的最有成就感，也是最有技术感和科技感的解谜游戏了。

[ Hack NET Mac版截图 ]

另外，游戏里面也自带了攻略让玩家查看，基本上语言关过了，完成游戏问题并不大。所以，大家别因为粗糙的命令行界面而错过这几款好游戏哦！

多平台支持的黑客游戏：

Hack Run、Hack Run Zero、Hack Time、Hack Net 这多部作品均提供了 iOS 和 Mac 版本。不过比较可惜的是，Android 平台上目前只推出了 Hack Run  和 Hack Run Zero，续作将会在稍后才能推出。所以，还是推荐大家用 iPad 或者 Mac 来玩这款游戏吧，不仅作品齐全，而且用大点的屏幕游玩体验更好。喜欢用脑的朋友快去试试这款游戏吧。
